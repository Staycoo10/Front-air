# Mini-Airbnb Clone (front)

A modern, responsive front-end clone of the Airbnb user interface, built to showcase modern web development practices, pixel-perfect design implementation, and seamless user experiences. 

## ✨ Features

* **Pixel-Perfect UI:** Closely replicates the modern Airbnb aesthetic, including typography, spacing, and brand colors.
* **Interactive Authentication UI:** A fully functional frontend login/signup flow with input formatting, country code selection, and state management.
* **Smooth Horizontal Carousels:** Custom-built "Adventures" and "Services" sections featuring snap-scrolling carousels with hidden scrollbars for a native app feel.
* **Fully Responsive:** Fluid layouts that adapt perfectly to mobile, tablet, and desktop screens.
* **Zero-Dependency Icons:** Utilized inline SVGs for all icons to optimize performance and reduce bundle size.

## 🛠️ Tech Stack

* **Framework:** React 18
* **Tooling:** Vite (for lightning-fast HMR and building)
* **Language:** TypeScript (TSX) for static typing and robust code.
* **Styling:** Tailwind CSS for rapid, utility-first UI development.
* **Routing:** React Router DOM for seamless Single Page Application (SPA) navigation.
